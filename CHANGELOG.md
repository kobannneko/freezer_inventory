# Changelog

## v2.1
- 初期表示を「表示」画面に変更
- 「表示」「入力」タブに変更
- ログイン／ログアウト欄を下部へ移動
- カテゴリを見直し（鶏肉・豚肉・牛肉・加工肉・魚介類・お菓子・調味料など）
- 数量を0.25刻みに対応
- 数量操作に `-1 / -0.25 / 数量直接入力 / +0.25 / +1` を追加
- 数量が0になった場合の削除確認を追加
- 古い順固定表示に変更
- 冷凍日からの経過日数を色分け表示
- カテゴリ別まとめを追加
- 最近登録したものから再登録できる機能を追加

## v2.0
- Firebase Authentication（Googleログイン）を追加
- Cloud Firestoreによる夫婦間リアルタイム共有に対応
- localStorageからFirestoreへの初回移行機能を追加
- HTML/CSS/JavaScriptを分離
